> **Axxon Digital** is a premium, high-performance web application designed to showcase digital marketing services, portfolios, and client success stories. It features smooth page transitions, interactive UI elements, and a fully responsive design tailored for a top-tier user experience.

---

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| 🎨 **Dynamic Portfolio** | Detailed case studies, project metrics, and filterable galleries. |
| 🌊 **Fluid Animations** | Powered by Motion for seamless page transitions and scroll reveals. |
| 📱 **Responsive Design** | Mobile-first approach ensuring perfect rendering across all devices. |
| ⚡ **Modern UI/UX** | Custom cursors, marquee text, and tilt cards for an engaging interface. |
| 🌙 **Theme Support** | Seamless theme switching (Light, Dark, and Custom accents). |

---

## 🛠️ Technology Stack

### Frontend Architecture
- **Core:** React 19
- **Language:** TypeScript
- **Routing:** React Router DOM v7
- **Build Tool:** Vite

### Styling & UI
- **Framework:** Tailwind CSS v4
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React
